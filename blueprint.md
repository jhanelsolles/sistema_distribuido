# Overview

This project is a Flutter application that receives push notifications from a Django backend using Firebase Cloud Messaging (FCM).

# Features

* **Firebase Integration:** The application is integrated with Firebase to enable push notifications.
* **Cross-platform:** The application is configured to receive notifications on both Android and web platforms.
* **Token Display:** The application displays the FCM registration token, which is essential for sending notifications to the device.
